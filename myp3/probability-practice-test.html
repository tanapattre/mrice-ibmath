MYP3 Probability Practice Test

Markscheme · 40 marks

Section A — Vocabulary & Concepts [15 marks]

A1 — Sample Space [3 marks]

(a) The sample space is the set of all possible outcomes of an experiment. A1
(b) $\{1, 2, 3, 4, 5, 6\}$ A1
(c) No / It is not in the sample space. A1

A2 — Outcome [3 marks]

(a) An outcome is a single possible result of an experiment. A1
(b) Red (or Green) A1
(c) Any valid real-life example, e.g. "Getting heads when tossing a coin" or "Rolling a 3 on a die." A1

A3 — Probability Notation [3 marks]

(a) $P(A)$ → The probability that event A happens
$P(A')$ → The probability that event A does not happen
$P(A \cap B)$ → The probability that event A and event B both happen A1
(b) "The probability of event A is 0.3" or "There is a 30% chance that event A happens." A1
(c) $P(\text{red}) = \frac{5}{10} = \frac{1}{2} = 0.5$ A1

A4 — Complementary Events [3 marks]

(a) Two events are complementary if one must happen and the other cannot; they cover all possibilities. A1
(b) "The probability that event A does not happen" or "The probability of the complement of A." A1
(c) $P(\text{not rain}) = 1 - 0.4 = 0.6$ A1

A5 — Mutually Exclusive Events [2 marks]

(a) Two events are mutually exclusive if they cannot happen at the same time. A1
(b) "Rolling a 2" and "Rolling an even number" — these CAN happen together (2 is even), so not mutually exclusive. Mutually exclusive pair: "Rolling a 2" and "Rolling a number greater than 4" (since 2 is not > 4). A1

A6 — Independent Events [2 marks]

(a) Two events are independent if the outcome of one does not affect the outcome of the other. A1
(b) Yes, they are independent. The result of the coin toss does not affect the result of the die roll. A1

A7 — Dependent Events [2 marks]

(a) Two events are dependent if the outcome of the first event affects the outcome of the second event. A1
(b) Yes, they are dependent. After removing the first marble, there are fewer marbles in the bag, so the probabilities for the second draw change. A1

A8 — Values of Probability [2 marks]

(a) Between 0 and 1 inclusive, or $0 \leq P(A) \leq 1$. A1
(b) (i) Mark at 0, (ii) Mark at 1. A1
(c) $0.5$ or $\frac{1}{2}$ A1

A9 — With and Without Replacement [2 marks]

(a) "With replacement" means the object is put back after it is drawn, so the probabilities stay the same for the next draw. A1
(b) $P(\text{red then red}) = \frac{2}{4} \times \frac{2}{4} = \frac{4}{16} = \frac{1}{4} = 0.25$ A1

Section B — Core Skills [25 marks]

Question 1 — Basic Probability [4 marks]

(a) $P(\text{red}) = \frac{4}{10} = \frac{2}{5} = 0.4$ A1
(b) $P(\text{not green}) = 1 - \frac{3}{10} = \frac{7}{10} = 0.7$ OR $\frac{4+3}{10} = \frac{7}{10}$ M1A1
(c) $P(\text{yellow}) = 0$ A1

Question 2 — Complementary Events [3 marks]

(a) $P(\text{not pass}) = 1 - 0.72 = 0.28$ A1
(b) $0.72 + 0.28 = 1.00$ AG M1A1

Question 3 — Listing Outcomes [3 marks]

(a) (H,1), (H,2), (H,3), (H,4), (T,1), (T,2), (T,3), (T,4) A1A1 (deduct 1 mark if any missing or incorrect)
(b) 8 outcomes A1

Question 4 — Two-Dimensional Sample Space [5 marks]

(a) Grid with 6×6 = 36 cells labelled correctly (Die 1 rows, Die 2 columns). A1A1A1
(b) Sums > 9: (4,6), (5,5), (5,6), (6,4), (6,5), (6,6) — 6 outcomes. $P = \frac{6}{36} = \frac{1}{6}$ M1A1

Question 5 — Tree Diagram (One Event) [4 marks]

(a) Tree diagram with two branches: Red ($\frac{2}{5}$) and Blue ($\frac{3}{5}$). Probabilities must sum to 1. A1A1
(b) $P(\text{red}) = \frac{2}{5} = 0.4$ A1
(c) $P(\text{blue}) = \frac{3}{5} = 0.6$ A1

Question 6 — Tree Diagram (Two Events) [6 marks]

(a) First set of branches: A ($\frac{1}{3}$), B ($\frac{1}{3}$), C ($\frac{1}{3}$). Second set of branches from each: A ($\frac{1}{3}$), B ($\frac{1}{3}$), C ($\frac{1}{3}$). All nine outcomes shown. A1A1A1
(b) $P(A \text{ then } B) = \frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$ M1A1
(c) $P(\text{same letter}) = P(AA) + P(BB) + P(CC) = \frac{1}{9} + \frac{1}{9} + \frac{1}{9} = \frac{3}{9} = \frac{1}{3}$ A1

Question 7 — Reasoning [3 marks]

(a) Any fair rule, e.g. "Player 1 wins if the die shows 1, 2, or 3. Player 2 wins if it shows 4, 5, or 6." A1
(b) Each player has 3 winning numbers out of 6, so $P(\text{win}) = \frac{3}{6} = \frac{1}{2}$ for each player. Equal probabilities = fair game. M1A1
