MYP3 Probability Practice Test — Version A

40 minutes · No calculator required · Write all working in the spaces provided

Instructions: Answer all questions. Show your working where required. All answers in this test should be given without a calculator. There are 44 marks available.
Question 1 [3 marks]

Write down what is meant by the term sample space.

A standard six-sided die is rolled. List all the outcomes in the sample space.

State whether "rolling a 7" is an outcome in this sample space.

Question 2 [4 marks]

A bag contains 4 red marbles, 3 blue marbles, and 3 green marbles. A marble is chosen at random.

(a) Calculate the probability that the marble is red. [1]

(b) Calculate the probability that the marble is not green. [2]

(c) State the probability that the marble is yellow. [1]

Question 3 [2 marks]

The probability that it rains tomorrow is $0.4$.

(a) Calculate the probability that it does not rain tomorrow.

(b) Show that the sum of these two probabilities is equal to 1.

Question 4 [3 marks]

A coin is tossed and a four-sided die (labelled 1, 2, 3, 4) is rolled.

(a) Write down all possible outcomes in the sample space. Use the format (coin, die). [2]

(b) Calculate the total number of outcomes. [1]

Question 5 [2 marks]

A coin is tossed and a die is rolled.

(a) State whether these two events are independent.

(b) Justify your answer.

Question 6 [4 marks]

A bag contains 2 red balls and 3 blue balls. A ball is chosen at random, its colour is recorded, and the ball is replaced.

(a) Draw a tree diagram to represent this experiment. Label all branches with their probabilities. [2]

(b) Calculate the probability of choosing a red ball. [1]

(c) Calculate the probability of choosing a blue ball. [1]

(a) Calculate the probability that the student does not pass the test.

Question 1 — Sample Space [3 marks]

(b) $\{1, 2, 3, 4, 5, 6\}$ A1

Question 2 — Basic Probability [4 marks]

(a) $P(\text{red}) = \frac{4}{10} = \frac{2}{5} = 0.4$ A1
(b) $P(\text{not green}) = 1 - \frac{3}{10} = \frac{7}{10} = 0.7$ OR $\frac{4+3}{10} = \frac{7}{10}$ M1A1
(c) $P(\text{yellow}) = 0$ A1

Question 3 — Complementary Events [2 marks]

(a) $P(\text{not rain}) = 1 - 0.4 = 0.6$ A1
(b) $0.4 + 0.6 = 1.00$ AG A1

Question 4 — Listing Outcomes [3 marks]

(a) (H,1), (H,2), (H,3), (H,4), (T,1), (T,2), (T,3), (T,4) A1A1 (deduct 1 if any missing/incorrect)
(b) 8 outcomes A1

Question 5 — Independent Events [2 marks]

(a) Yes, they are independent. A1
(b) The result of the coin toss does not affect the result of the die roll. A1

Question 6 — Tree Diagram (One Event) [4 marks]

(a) Two branches: Red ($\frac{2}{5}$) and Blue ($\frac{3}{5}$). Probabilities sum to 1. A1A1
(b) $\frac{2}{5} = 0.4$ A1
(c) $\frac{3}{5} = 0.6$ A1

Question 7 — Probability Notation [3 marks]

(a) $P(A)$ → The probability that event A happens; $P(A')$ → does not happen; $P(A \cap B)$ → A and B both happen A1
(b) "The probability of event A is 0.3" A1
(c) $\frac{5}{10} = \frac{1}{2} = 0.5$ A1

Question 8 — 2D Sample Space [5 marks]

(a) 6×6 grid with all 36 outcomes. Correct axes labels. A1A1A1
(b) Sums > 9: (4,6), (5,5), (5,6), (6,4), (6,5), (6,6) — 6 outcomes. $P = \frac{6}{36} = \frac{1}{6}$ M1A1

Question 9 — Values of Probability [2 marks]

(a) $0 \leq P(A) \leq 1$ or "between 0 and 1 inclusive" A1
(b) (i) Mark at 0, (ii) Mark at 1 A1

Question 10 — Tree Diagram (Two Events) [6 marks]

(a) First branches: A ($\frac{1}{3}$), B ($\frac{1}{3}$), C ($\frac{1}{3}$). Second set same. 9 outcomes shown. A1A1A1
(b) $\frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$ M1A1
(c) $P(AA) + P(BB) + P(CC) = \frac{1}{9} + \frac{1}{9} + \frac{1}{9} = \frac{1}{3}$ A1

Question 11 — Mutually Exclusive [2 marks]

(a) Two events are mutually exclusive if they cannot happen at the same time. A1
(b) "Rolling a 2" and "Rolling a number greater than 4" (2 is not > 4, so mutually exclusive) A1

Question 12 — Complementary Events (Test) [3 marks]

(a) $1 - 0.72 = 0.28$ A1
(b) $0.72 + 0.28 = 1.00$ AG M1A1

Question 13 — With/Without Replacement [2 marks]

(a) The object is put back after it is drawn, so probabilities stay the same. A1
(b) $\frac{2}{4} \times \frac{2}{4} = \frac{4}{16} = \frac{1}{4} = 0.25$ A1

Question 14 — Outcome [2 marks]

(a) An outcome is a single possible result of an experiment. A1
(b) Any valid example, e.g. "Getting heads when tossing a coin." A1

Question 15 — Dependent Events [2 marks]

(a) Two events are dependent if the outcome of the first affects the outcome of the second. A1
(b) Yes, dependent. After removing the first marble, fewer remain, so probabilities change. A1

Question 16 — Reasoning [3 marks]

(a) Any fair rule, e.g. "Player 1 wins on 1,2,3; Player 2 wins on 4,5,6." A1
(b) Each player has 3 winning numbers: $P = \frac{3}{6} = \frac{1}{2}$. Equal probabilities = fair. M1A1
