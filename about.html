About This Site

IB Mathematics teaching resources for DP and MYP

Purpose

This site hosts scaffolded practice worksheets, lesson materials, and reference documents for IB DP Mathematics: Applications & Interpretation SL and MYP3 Mathematics.

How to Use

  • Students: Open any worksheet in "Student View" to see questions only. Print or work digitally.
  • Teachers: Toggle to "Teacher View" to reveal full markschemes with M1/A1/R1 marks.
  • Print: Use browser print (Ctrl/Cmd + P) — the site is optimized for printing.
  • Math display: Equations render using MathJax (LaTeX) — works on any modern browser.

Structure

  • DP AI SL — Probability Distributions unit (Syllabus 4.7 & 4.8) with 6 scaffolded worksheets
  • MYP3 — Unit 4 Probability with study guides, tests, and enrichment investigations
  • References — IB official documents and publisher textbooks

Technical

Built with static HTML, CSS, and MathJax for equation rendering. Hosted on GitHub Pages. No server-side code needed.
